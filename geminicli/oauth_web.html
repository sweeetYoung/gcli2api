Google OAuth 认证

⚠️ 重要：请先启用以下 API 服务

在开始认证前，请确保为您的项目启用以下API：

  1. Gemini Cloud Assist API
    点击这里启用 →
  2. Cloud AI Companion API
    点击这里启用 →

注意：如果未启用这些API，OAuth认证过程将会失败。

步骤一：认证链接

点击此链接进行 OAuth 认证

认证步骤：

  1. 点击上方认证链接，在新窗口中完成 Google 登录
  2. 授权应用访问您的 Google Cloud 项目
  3. 看到 "OAuth authentication successful!" 页面后，关闭该窗口
  4. 返回本页面，点击下方"获取认证文件"按钮

步骤二：认证成功！
